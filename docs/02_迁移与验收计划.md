# Excel 迁移与验收计划

## 迁移原则

所有导入数据都应记录来源文件、Sheet 和行号。先生成导入草稿，再进行人工复核；不得直接把工作簿备注中的敏感信息复制进应用。

## Sheet 映射

| 源 Sheet | 目标对象 | 自动导入 | 人工复核 |
|---|---|---|---|
| 账户余额 | 账户和余额快照 | 项目、分类、金额、统计日、流动性 | 机构、归属人、币种、可用性、重复账户 |
| 年度收入 | 预计现金流（流入） | 流动资金性质、日期/月度、项目、金额 | 资产性质项目改为内部转账或估值，不算收入 |
| 年度支出 | 预计现金流（流出） | 流动资金性质、日期/月度、项目、金额 | 房贷空白月份、一次性资产转换、实际支付账户 |
| 资金预测 | 情景与校验 | 手工设置的生活、教育、旅行与调整项 | 月份、公式、余额滚动和期初差异不直接复用 |
| 固定资产和股票期权 | 资产、估值、负债、期权计划 | 名称、类别、状态、数量×单价可得估值 | 权益比例、估值日期、贷款余额、归属与行权信息 |
| 银行账户和证照信息 | 证照/账户元数据 | 证照类型、到期日、状态 | 完整标识、机构、归属、永久有效标识；秘密迁出 |

## 必须处理的复核队列

1. 账户余额总额与资金预测基线的 20,000 元差异。
2. 存单到期、保险缴费等“现金与资产同时变化”记录：按内部转账配对，不能同时记收入和支出。
3. 月度房贷及水电的空白金额，确认是否应按同额持续发生。
4. 房产和共有权益的持有比例、估值日期、估值来源、关联贷款余额。
5. 租约收入的合同总额、合同年限和应收/实收规则。
6. 期权的授予日、归属批次、行权价、到期日、税费与实际持有人。
7. 所有证照的到期日或“长期有效”标志；已过期和 90 天内到期项目优先处理。

## 安全迁移助手

`web/tools/import-workbook.mjs` 只提取非敏感业务字段，默认将导入草稿写入被 Git 忽略的 `web/.local-import/`。它不会读取或导出备注、账号/卡号、证件号、地址、密码、验证码、邮箱或附件路径。

导入后应用会提示被跳过的资产变化和异常行数。所有跳过项必须经过人工分类后才可进入正式月结。

## 上线质量闸门

- 账户快照合计与核准期初余额一致，或每个差异有解释。
- 所有内部转账有来源、去向、日期、金额和相同币种。
- 资产净值不包含未确认权益或未连接的负债余额。
- 所有预测项目标记为待发生、已发生或取消。
- 所有适用证照都有有效到期日或永久有效标志。
- 导入草稿和应用数据库中不存在明文秘密。
- 导出 JSON 能在空浏览器配置中恢复并得到相同记录数。

## 回退

每次导入前先导出 JSON 备份。导入错误时，在“数据安全”中恢复最近备份；不要通过手工修改导入草稿绕过复核，避免丢失来源行和审计依据。
