# 家庭资产与资金管理系统

本仓库包含两部分：

- `web/`：可运行的本地优先 MVP，支持账户、流水、内部转账、资产负债、现金流预测、证照提醒与 JSON 备份恢复。
- `docs/` 与 `db/`：将现有工作簿迁移为可审计系统的需求、迁移规则和 SQLite 升级数据模型。

## 当前使用方式

进入 [web/README.md](web/README.md) 按说明启动本地应用。应用默认加载脱敏示例数据；你的真实数据可先通过 `web/tools/import-workbook.mjs` 生成安全导入草稿，再通过界面导入。

## 设计原则

1. 账户余额、资产估值和负债余额分别维护，避免重复计算净资产。
2. 收入、支出和内部转账严格区分；内部转账不会被算作家庭收入或支出。
3. 实际流水与未来预测分离；预测发生后应由实际流水闭环。
4. 不保存密码、PIN、CVV、验证码、恢复码、完整账号或完整证件号。
5. 本地数据要定期导出备份，并存放在受加密保护的位置。
