# 家财管家（本地 MVP）

这是一个浏览器内运行的家庭资产与资金管理原型。它覆盖账户余额、收入/支出/内部转账、资产与负债、预计现金流、证照提醒，以及本地备份恢复。

## 本地运行

在本目录运行：

```powershell
pnpm run dev
```

然后打开终端显示的本地地址。构建校验使用：

```powershell
pnpm run build
pnpm exec node --test tests\app-source.test.mjs
```

## 数据边界

- 业务数据保存在当前浏览器的 IndexedDB，不会自动上传。
- 通过界面的“导出备份”创建 JSON，本地“导入备份”恢复。
- 不要录入完整账号、证件号、密码、PIN、CVV、验证码或恢复码；仅可保存外部保险库引用。
- 浏览器清理数据会删除本地记录，因此应定期导出备份并放到加密位置。

## 从现有 Excel 迁移

`tools/import-workbook.mjs` 会安全生成导入草稿，只提取非敏感业务字段；它不会读取备注或标识信息。详细说明见 [tools/README.md](tools/README.md)。

导入后，请优先复核内部资产转换、期初余额差异、房贷规则、资产权益比例、负债余额及证照期限。

## 后续升级

当需要多设备协作、加密附件或多人权限时，可基于上一级目录的 [SQLite 数据模型](../db/schema.sql) 升级为本地服务或受控服务端。当前 MVP 仍保持本地优先，避免把家庭金融信息默认发送到云端。
